# AutoCooldown

```
Default Keybind: NONE
Category: Combat
```

AutoCooldown automatically releases items that have a charge or use duration (such as bows, crossbows, and tridents) when held down, then re-uses them after the cooldown.

## Detectable By
- [ ] NoCheatPlus
- [ ] Vulcan
- [ ] AdvancedAntiCheat
- [ ] Verus
- [ ] Grim
- [ ] Matrix
- [ ] Negativity
- [ ] Karhu
- [ ] Buzz

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Use % | Slider | Charge percentage at which the item is released (25 - 100, Step: 5) | 100 |
| Randomness | Slider | Randomness of the charge percentage and re-use delay. Used to simulate differences in player input (0.0 - 10.0, Step: 1.0) | 2.0 |
