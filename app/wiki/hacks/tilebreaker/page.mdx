# TileBreaker

```
Default Keybind: NONE
Category: World
```

TileBreaker automatically destroys fragile blocks like torches, flowers, crops, and tall grass around you within a
configurable radius. A colored highlight box is shown around each targeted block.

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Color | Color | The color of the highlight box rendered around targeted blocks | Cyan (0.0, 1.0, 1.0) |
| Radius | Slider | The radius (in blocks) around the player within which TileBreaker will scan for and destroy blocks (0.0 - 15.0, Step: 1.0) | 5.0 |
