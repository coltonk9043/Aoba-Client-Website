# FastBreak

```
Default Keybind: NONE
Category: Misc
```

FastBreak lets you mine and break blocks faster than normal using a configurable speed multiplier.
It can also remove the mining slowdown from being underwater.

## Detectable By
- [x] NoCheatPlus
- [ ] Vulcan
- [x] AdvancedAntiCheat
- [ ] Verus
- [x] Grim
- [x] Matrix
- [ ] Negativity
- [ ] Karhu
- [ ] Buzz

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Multiplier | Slider | Multiplier for how fast blocks will break. Higher values break blocks faster (1.0 - 10.0, Step: 0.05) | 1.25 |
| Ignore Water | Toggle | Ignores the mining slowdown caused by being submerged in water | off |
