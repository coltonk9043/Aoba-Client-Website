# AutoOminous

```
Default Keybind: NONE
Category: MISC
```

AutoOminous automatically drinks an Ominous Bottle from your hotbar when a raid ends, allowing you to chain raids back-to-back without manual intervention.

## Settings

| Name | Type | Description | Default |
|------|------|-------------|---------|
| Swap Back | Boolean | Whether your selected hotbar slot will be switched back to the slot you had selected before drinking the Ominous Bottle | true |
