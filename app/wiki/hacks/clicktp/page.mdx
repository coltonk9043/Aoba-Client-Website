# ClickTP

```
Default Keybind: NONE
Category: Movement
```

ClickTP allows you to instantly teleport to any block you are looking at by right-clicking.
Simply aim at a block within range and right-click to teleport there.

## Settings

| Name | Type | Description | Default |
|------|------|-------------|---------|
| Max Distance | Float | The maximum distance (in blocks) the player can teleport (1.0 - 200.0, Step: 1.0) | 10.0 |
