# BedAura

```
Default Keybind: NONE
Category: Combat
```

BedAura automatically finds and destroys nearby beds within a configurable radius, making it ideal for Bed Wars.
The targeted bed is highlighted with a colored outline, and a "legit" mode is available to reduce the chance of anti-cheat detection.

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Radius | Slider | The search radius around the player for detecting beds (0 - 15) | 5 |
| Legit | Toggle | When enabled, uses a raycast to ensure the player is looking at the bed before breaking it, making the behavior appear more natural | Off |
| Rotation Mode | Enum | Controls how the player's view rotates toward the target bed | NONE |
| Max Rotation | Slider | The maximum speed at which the player's view will rotate toward the target bed (1 - 360) | 10 |
| Yaw Rotation Jitter | Slider | Adds randomness to the player's horizontal (yaw) rotation for a more human-like appearance (0 - 10) | 0 |
| Pitch Rotation Jitter | Slider | Adds randomness to the player's vertical (pitch) rotation for a more human-like appearance (0 - 10) | 0 |
| Color | Color | The color of the 3D box rendered around the targeted bed block | Cyan (0, 1, 1) |
