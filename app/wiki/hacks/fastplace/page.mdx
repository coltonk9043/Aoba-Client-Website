# FastPlace

```
Default Keybind: NONE
Category: Misc
```

FastPlace lets you place blocks much faster than normal by reducing or removing the delay between placements.

## Detectable By
- [x] NoCheatPlus
- [ ] Vulcan
- [x] AdvancedAntiCheat
- [ ] Verus
- [ ] Grim
- [x] Matrix
- [x] Negativity
- [ ] Karhu
- [ ] Buzz

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Delay | Slider | The delay between block placements in ticks. A value of 0 removes the delay entirely for the fastest placement speed (0.0 - 5.0, Step: 1.0) | 0.0 |
