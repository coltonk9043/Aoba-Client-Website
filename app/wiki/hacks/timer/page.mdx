# Timer

```
Default Keybind: NONE
Category: Misc
```

Timer speeds up or slows down the game. Values above 1.0 make you move, mine, and interact faster, while values below
1.0 slow everything down. This is most effective in singleplayer but works on servers too.

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Multiplier | Slider | The multiplier that affects the game speed. Values above 1.0 speed up the game, values below 1.0 slow it down (0.1 - 15.0, Step: 0.1) | 1.0 |
