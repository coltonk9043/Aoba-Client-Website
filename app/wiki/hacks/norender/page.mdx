# NoRender

```
Default Keybind: NONE
Category: Render
```

NoRender lets you selectively hide annoying screen effects like the fire overlay, 
pumpkin overlay, portal effect, and more, giving you a clearer view of your surroundings.

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Eating Particles | Boolean | Does not render eating particles | false |
| Totem Animation | Boolean | Does not render the totem floating animation | false |
| Vignette | Boolean | Does not render the Minecraft vignette | false |
| Pumpkin Overlay | Boolean | Does not render the pumpkin overlay when you are wearing one | false |
| Fire Overlay | Boolean | Does not render the overlay when the player is on fire | false |
| Portal Overlay | Boolean | Does not render the overlay when travelling through a portal | false |
| Powder Snow Overlay | Boolean | Does not render the overlay when in powder snow | false |
| Liquid Overlay | Boolean | Does not render the overlay when in liquids | false |
| Crosshair | Boolean | Does not render the crosshair | false |
