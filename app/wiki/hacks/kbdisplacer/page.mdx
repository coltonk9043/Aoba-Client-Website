# KBDisplacer

```
Default Keybind: NONE
Category: Combat
```

KBDisplacer (Knockback Displacer) briefly spoofs your rotation when attacking while sprinting, redirecting the knockback you take from the hit in a different direction.

## Detectable By
- [ ] NoCheatPlus
- [ ] Vulcan
- [ ] AdvancedAntiCheat
- [ ] Verus
- [ ] Grim
- [ ] Matrix
- [ ] Negativity
- [ ] Karhu
- [ ] Buzz

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Angle | Slider | The angle by which the knockback is displaced (-180.0 - 180.0, Step: 1.0) | 180.0 |
| Angle randomization | Slider | Displacement angle randomization (0.0 - 180.0, Step: 1.0) | 0.0 |
