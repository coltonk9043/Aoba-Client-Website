# MaceAura

```
Default Keybind: NONE
Category: Combat
```

MaceAura automatically attacks nearby entities with the Mace, simulating a fall from a great height to deal massive
bonus damage. For best results, enable NoFall alongside this module.

## Detectable By
- [x] NoCheatPlus
- [x] Vulcan
- [x] AdvancedAntiCheat
- [x] Verus
- [x] Grim
- [x] Matrix
- [x] Negativity
- [x] Karhu
- [x] Buzz

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Radius | Slider | Controls the maximum distance at which MaceAura will detect and target entities (0.1 - 10.0, Step: 0.1) | 5.0 |
| Height | Slider | Determines how high the position spoof will be, simulating a fall from that height. Higher values result in more damage dealt by the Mace (1.0 - 255.0) | 100.0 |
| Target Animals | Toggle | When enabled, MaceAura will target passive animal mobs | Off |
| Target Monsters | Toggle | When enabled, MaceAura will target hostile monster mobs | On |
| Target Players | Toggle | When enabled, MaceAura will target other players | On |
| Target Friends | Toggle | When enabled, MaceAura will also target players on your friends list. When disabled, friends are excluded from targeting | Off |
