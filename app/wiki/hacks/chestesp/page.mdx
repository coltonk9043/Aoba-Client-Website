# ChestESP

```
Default Keybind: NONE
Category: RENDER
```

ChestESP will draw boxes, which are visible through walls, on any Chest. 

This lets you find bases, and hidden chests that you may have not been able to see otherwise.

# Settings
 - Color (Color): If Rainbow is not enabled, the color that the boxes will be rendered.
 - Rainbow (Toggle): Whether or not Rainbow color mode is enabled.
 - Effect Speed (Variable): The speed at which Rainbow will transition to the next color.
