# DiscordRPC

```
Default Keybind: NONE
Category: MISC
```

DiscordRPC toggles Discord Rich Presence integration on and off. When enabled, it starts the Discord RPC service to
display Aoba Client activity on your Discord profile. When disabled, the RPC service is stopped and the presence is cleared.
