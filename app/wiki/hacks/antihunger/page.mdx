# AntiHunger

```
Default Keybind: NONE
Category: MISC
```

AntiHunger slows down your hunger depletion, letting you go longer without eating. It will not activate while riding, swimming, or underwater.

## Settings

| Name | Type | Description | Default |
|------|------|-------------|---------|
| Sprint | Boolean | Controls whether sprint packets are modified. When enabled, sprint start packets are cancelled to reduce server-side hunger drain | true |
| On Ground | Boolean | Controls whether the player's on-ground status is spoofed in movement packets. When enabled, the on-ground flag is set to false to reduce hunger consumption | true |
