# XRay

```
Default Keybind: X
Category: Render
```

XRay hides all blocks except those on a configurable whitelist, letting you see ores and other
valuable blocks through solid terrain. The default list includes all common ores and their deepslate variants.

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Blocks | Blocks | The set of blocks that remain visible when XRay is active | Emerald Ore, Emerald Block, Diamond Ore, Diamond Block, Gold Ore, Gold Block, Iron Ore, Iron Block, Coal Ore, Coal Block, Redstone Block, Redstone Ore, Lapis Ore, Lapis Block, Nether Quartz Ore, Mossy Cobblestone, Stone Bricks, Oak Planks, Deepslate Emerald Ore, Deepslate Diamond Ore, Deepslate Gold Ore, Deepslate Iron Ore, Deepslate Coal Ore, and Ancient Debris |
| Show Fluids | Boolean | Whether fluids (water and lava) remain visible when XRay is active | true |
