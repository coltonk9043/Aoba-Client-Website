# NoSlowdown

```
Default Keybind: NONE
Category: Movement
```

NoSlowdown prevents you from being slowed down by blocks like Soul Sand, Cobwebs, and similar.
You can remove the slowdown entirely or reduce it by a configurable amount.

## Detectable By
- [x] NoCheatPlus
- [x] Vulcan
- [x] AdvancedAntiCheat
- [ ] Verus
- [x] Grim
- [x] Matrix
- [ ] Negativity
- [x] Karhu

## Settings

| Name | Type | Description | Default |
|------|------|-------------|---------|
| Multiplier | Float | Controls the walk speed multiplier when moving through blocks that slow the player down. A value of 0 completely removes the slowdown effect, while higher values apply a proportional reduction to the slowdown (0.0 - 1.0, Step: 0.1) | 0.0 |
