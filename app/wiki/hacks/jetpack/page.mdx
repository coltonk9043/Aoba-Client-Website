# Jetpack

```
Default Keybind: NONE
Category: Movement
```

Jetpack launches you upward while you hold the jump key, complete with flame particle effects.
Unlike Fly, you arc and fall naturally when you release the key. It also works while riding entities.

## Settings

| Name | Type | Description | Default |
|------|------|-------------|---------|
| Speed | Float | Controls the upward speed of the jetpack (0.1 - 5.0, Step: 0.1) | 0.5 |
| Thruster Particle Amount | Integer | Number of flame particles generated by each thruster per tick (1 - 20, Step: 1) | 1 |
| Jump Motion Y | Float | Upward motion applied when jump is pressed while riding an entity (0.1 - 2.0, Step: 0.1) | 0.3 |
| Thruster Spread | Float | Controls the horizontal spread between the two thruster particle emitters (0.1 - 1.0, Step: 0.1) | 0.25 |
