# BowAimbot

```
Default Keybind: NONE
Category: Combat
```

BowAimbot automatically aims your bow or crossbow at the nearest target, accounting for arrow arc and target movement to land accurate shots.
It also enables the Trajectory module to show you the projected arrow path while aiming.

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Target Mobs | Toggle | When enabled, the aimbot will target mobs | OFF |
| Target Players | Toggle | When enabled, the aimbot will target players | ON |
| Ticks | Slider | Controls how frequently the aimbot recalculates its aim, measured in ticks. Lower values update more often but may cause more lag (1 - 20) | 1 |
| Prediction | Slider | Sets the strength of BowAimbot's movement prediction, allowing it to lead shots on moving targets. Higher values predict further ahead (0 - 10) | 2 |
