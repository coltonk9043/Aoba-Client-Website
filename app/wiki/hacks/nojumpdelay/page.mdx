# NoJumpDelay

```
Default Keybind: NONE
Category: Movement
```

NoJumpDelay removes or reduces the cooldown between jumps, letting you jump again much more quickly after landing.
This is useful for parkour, PvP strafing, and any situation where rapid successive jumps are needed.

## Settings

| Name | Type | Description | Default |
|------|------|-------------|---------|
| Delay | Integer | Controls the jump cooldown in ticks. Setting it to 0 removes the jump delay entirely, while higher values allow a longer cooldown between jumps (0 - 20) | 1 |
