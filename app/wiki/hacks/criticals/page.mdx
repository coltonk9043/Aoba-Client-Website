# Criticals

```
Default Keybind: NONE
Category: Combat
```

Criticals forces every melee attack to deal critical hit damage (50% bonus), even while standing on the ground.
A "Legit" mode is available that performs an actual jump before each hit for a more natural appearance.

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Legit | Toggle | When enabled, the player performs a real jump to trigger the critical hit, which is more legitimate but visually obvious. 
When disabled, the module silently spoofs position packets to fake the fall condition | off |
