# FastLadder

```
Default Keybind: NONE
Category: Movement
```

FastLadder lets you climb ladders and vines much faster than normal. Moving forward into the ladder gives you the fastest climb speed,
while sideways movement results in a slower ascent.

## Settings

| Name | Type | Description | Default |
|------|------|-------------|---------|
| Speed | Float | Base climbing speed applied when ascending ladders (0.1 - 0.5, Step: 0.1) | 0.2 |
| Acceleration Boost | Float | Extra speed added on top of the base speed when moving upwards on ladders (0.01 - 0.2, Step: 0.01) | 0.08 |
| Deceleration Penalty | Float | Speed reduction subtracted from the total climb speed when providing only sideways input without forward movement (0.01 - 0.2, Step: 0.01) | 0.08 |
