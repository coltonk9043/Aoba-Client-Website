# FocusFPS

```
Default Keybind: NONE
Category: Render
```

FocusFPS limits the frame rate of the game when the game window is not in focus. This helps reduce CPU and GPU usage
while the game is running in the background, saving system resources and lowering power consumption.

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| FPS | Float | The target frames per second when the game window is not in focus. Min: 1, Max: 45, Step: 1 | 30 |
