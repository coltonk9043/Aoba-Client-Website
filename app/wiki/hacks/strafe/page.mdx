# Strafe

```
Default Keybind: NONE
Category: Movement
```

Strafe gives you much better control over your movement while in the air, allowing for sharper turns and direction changes
than normally possible.

- [x] Vulcan

## Settings

| Name | Type | Description | Default |
|------|------|-------------|---------|
| Intensity | Float | Controls the strafe intensity, which affects how much additional speed is applied during air movement (0.0 - 0.3, Step: 0.1) | 0.1 |
