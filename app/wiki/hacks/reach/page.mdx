# Reach

```
Default Keybind: NONE
Category: Combat
```

Reach extends the distance at which you can hit players and mobs, letting you land attacks before opponents can reach you.

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Distance | Slider | The distance, in blocks, that you can reach (1.0 - 128.0, Step: 1.0) | 5.0 |
