# Tooltips

```
Default Keybind: NONE
Category: Render
```

Tooltips adds enhanced item previews in your inventory. Hover over shulker boxes or bundles to see their contents, or hover over maps to see a preview of the map image.

## Detectable By
- [ ] NoCheatPlus
- [ ] Vulcan
- [ ] AdvancedAntiCheat
- [ ] Verus
- [ ] Grim
- [ ] Matrix
- [ ] Negativity
- [ ] Karhu
- [ ] Buzz

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Storage | Boolean | Renders the contents of the storage item | true |
| Maps | Boolean | Renders a map preview when hovering over a map item | true |
