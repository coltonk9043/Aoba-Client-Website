# EXPThrower

```
Default Keybind: NONE
Category: Misc
```

EXPThrower automatically throws experience bottles from your hotbar at a configurable speed and angle.
This is useful for quickly repairing Mending-enchanted gear or gaining experience levels.

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Pitch | Slider | The pitch angle for throwing XP bottles. A value of 90 aims straight down (0.0 - 90.0, Step: 1.0) | 90.0 |
| Auto Swap | Toggle | Automatically swap to XP bottles if not in hand | on |
| Throw Delay | Slider | Delay between throws in ticks. Lower values throw faster (1.0 - 100.0, Step: 1.0) | 20.0 |
