# AutoEat

```
Default Keybind: NONE
Category: MISC
```

Aimbot is a module that automatically looks towards the nearest entity and locks your crosshair to them.
It can be useful in situations where you must be constantly looking at a player, such as KitPvP.

# Settings
 - Hunger (Variable): The hunger at which AutoEat will trigger (measured in half-hungers).
