# EntityControl

```
Default Keybind: NONE
Category: Movement
```

EntityControl allows you to ride and steer horses, donkeys, mules, llamas, and other horse-type entities without needing a saddle.
This is useful in survival when you have found or bred a horse but do not have a saddle available.
