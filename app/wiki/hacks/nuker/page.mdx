# Nuker

```
Default Keybind: NONE
Category: World
```

Nuker automatically destroys all blocks around you within a configurable radius. In Creative mode,
it breaks all blocks simultaneously for maximum speed. In Survival mode, it breaks blocks one at a time.
You can blacklist specific blocks to skip them, and a highlight box shows the block currently being broken.

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Creative | Toggle | Enables creative mode destruction, which breaks all blocks in range simultaneously by sending instant break packets | off |
| Color | Color | The color of the highlight box rendered around the block currently being broken | Cyan (0.0, 1.0, 1.0) |
| Radius | Slider | The radius (in blocks) around the player within which Nuker will destroy blocks (0.0 - 15.0, Step: 1.0) | 5.0 |
| Blacklist | Blocks | A list of blocks that Nuker will skip and not break | empty |
