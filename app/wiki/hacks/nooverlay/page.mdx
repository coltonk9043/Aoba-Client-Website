# NoRender

```
Default Keybind: NONE
Category: Render
```

NoRender lets you selectively hide annoying screen effects like the fire overlay, pumpkin overlay,
portal effect, and more, giving you a clearer view of your surroundings.

## Detectable By
- [ ] NoCheatPlus
- [ ] Vulcan
- [ ] AdvancedAntiCheat
- [ ] Verus
- [ ] Grim
- [ ] Matrix
- [ ] Negativity
- [ ] Karhu
- [ ] Buzz

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Eating Particles | Boolean | Does not render eating particles | - |
| Totem Animation | Boolean | Does not render the totem floating animation | - |
| Vignette | Boolean | Does not render the Minecraft vignette | - |
| Pumpkin Overlay | Boolean | Does not render the pumpkin overlay when you are wearing one | - |
| Fire Overlay | Boolean | Does not render the overlay when the player is on fire | - |
| Portal Overlay | Boolean | Does not render the overlay when travelling through a portal | - |
| Powder Snow Overlay | Boolean | Does not render the overlay when in powder snow | - |
| Liquid Overlay | Boolean | Does not render the overlay when in liquids | - |
| Crosshair | Boolean | Does not render the crosshair | - |
