# SpawnerESP

```
Default Keybind: NONE
Category: Render
```

SpawnerESP has been removed and replaced by [BlockESP](/wiki/hacks/blockesp), which provides the same spawner highlighting functionality and more. 
BlockESP defaults to highlighting spawner blocks and can be configured to highlight any block type.
