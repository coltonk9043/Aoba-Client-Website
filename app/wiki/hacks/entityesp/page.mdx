# EntityESP

```
Default Keybind: NONE
Category: RENDER
```

EntityESP will draw boxes, which are visible through walls, on any player or entity within the render distance of the game.

# Settings
 - Passive Color (Color): If Rainbow is not enabled, the color of the box that passive entities will have.
 - Enemy Color (Color): If Rainbow is not enabled, the color of the box that enemy entities will have.
 - Misc Color (Color): If Rainbow is not enabled, the color of the box that misc entities will have.
 - Rainbow (Toggle): Whether or not Rainbow color mode is enabled.
 - Effect Speed (Variable): The speed at which Rainbow will transition to the next color.
