# BlockESP

```
Default Keybind: NONE
Category: Render
```

BlockESP highlights specific block types through walls, making them visible from any distance. You can configure which blocks to search for,
making it easy to locate ores, spawners, or any other block type across large areas.

## Detectable By
- [ ] NoCheatPlus
- [ ] Vulcan
- [ ] AdvancedAntiCheat
- [ ] Verus
- [ ] Grim
- [ ] Matrix

- [ ] Karhu
- [ ] Buzz

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Blocks | Blocks | The set of block types that BlockESP will search for and highlight | Spawner |
| Color | Color | The color used to render the ESP boxes around detected blocks | Cyan (R: 0, G: 255, B: 255, A: 76) |
| Line Thickness | Float | Adjusts the thickness of the ESP box lines. Min: 0.0, Max: 5.0 | 2.0 |
