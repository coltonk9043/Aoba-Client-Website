# Nametags

```
Default Keybind: NONE
Category: Combat
```

Nametags enlarges entity nametags so they are easier to read at a distance. You can optionally limit this to
players only and force nametags to always be visible regardless of distance or line of sight.

## Detectable By
- [ ] NoCheatPlus
- [ ] Vulcan
- [ ] AdvancedAntiCheat
- [ ] Verus
- [ ] Grim
- [ ] Matrix
- [ ] Negativity
- [ ] Karhu
- [ ] Buzz

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Scale | Slider | Controls the scale of the nametags (0 - 5, Step: 0.25) | 1.25 |
| Only Players | Toggle | Whether nametags are only enlarged for players | Off |
| Always Visible | Toggle | Whether nametags will always be displayed | Off |
