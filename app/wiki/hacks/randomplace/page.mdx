# RandomPlace

```
Default Keybind: NONE
Category: Misc
```

RandomPlace swaps to a random block from your hotbar each time you place a block,
creating varied and natural-looking builds. You can choose which hotbar slots are used for selection.

## Detectable By
- [ ] NoCheatPlus
- [ ] Vulcan
- [ ] AdvancedAntiCheat
- [ ] Verus

- [ ] Matrix

- [ ] Karhu

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Allowed Slots | Hotbar | Selects which slots of the hotbar should be used for random block selection. Each of the 9 hotbar slots can be individually enabled or disabled | all slots disabled |
| Must Hold Block | Toggle | When enabled, the module only activates when you are currently holding a block in your hand | on |
