# MCA

```
Default Keybind: NONE
Category: Misc
```

MCA (Middle Click Action) assigns a custom action to the middle mouse button. In Friend mode,
middle-click a player to add or remove them from your friends list. In Pearl mode, it throws an Ender Pearl.
In Firework mode, it launches a Firework Rocket, which is useful for Elytra flying.
