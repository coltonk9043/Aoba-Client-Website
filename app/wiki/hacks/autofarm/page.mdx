# AutoFarm

```
Default Keybind: NONE
Category: World
```

AutoFarm automatically plants, fertilizes, and harvests crops around you. It will use bone meal on growing crops, replant empty farmland, and harvest fully grown crops within a configurable radius.

- [ ] Vulcan
- [ ] AdvancedAntiCheat

- [ ] Matrix

- [ ] Karhu
- [ ] Buzz

## Settings
| Name | Type | Description | Default |
|------|------|-------------|---------|
| Radius | Slider | The radius (in blocks) around the player that AutoFarm will scan for crops and farmland (0.0 - 15.0, Step: 1.0) | 5.0 |
