# Spider

```
Default Keybind: NONE
Category: Movement
```

Spider lets you climb up any wall just by walking into it, similar to how spiders climb in Minecraft.

## Settings

| Name | Type | Description | Default |
|------|------|-------------|---------|
| Speed | Float | Controls the speed at which the player climbs up blocks (0.05 - 1.0, Step: 0.05) | 0.1 |
