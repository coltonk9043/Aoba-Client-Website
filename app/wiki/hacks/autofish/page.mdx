# AutoFish

```
Default Keybind: NONE
Category: MISC
```

AutoFish automatically reels in and recasts your fishing rod when a fish bites. It can switch to a fishing rod in your hotbar when enabled and optionally turn itself off if no rod is found.

## Settings

| Name | Type | Description | Default |
|------|------|-------------|---------|
| Auto Switch | Boolean | Automatically switches to a fishing rod in your hotbar before casting | true |
| Auto Toggle | Boolean | Automatically toggles the module off if no fishing rod is found in the hotbar | false |
