# ClickGUI

The ClickGUI is the central interface for Aoba Client. Nearly every function of the mod can be accessed through it, 
such as: enabling and disabling modules, adjusting their settings, adding HUD overlays, and more.

## Opening the ClickGUI

As of 1.21.11+, the default keybind to toggle the ClickGUI is **Right Shift**. If you would like to use a different key, 
the keybind can be changed in the HUD settings page (described below), or by running the following command in chat:

```
.aoba clickgui set [key]
.aoba clickgui set RSHIFT
```

You can rebind it at any time.

## Window System

The ClickGUI features a flexible window system. Each panel can be freely moved and resized to suit your layout.

- **Dragging** - Click anywhere along the top of a window and drag to reposition it.
- **Resizing** - Hover over the edge of a window until the cursor changes, then drag to resize. Windows have a minimum size and cannot be shrunk past it.
- **Persistence** - Window positions and sizes are saved across game sessions, so your layout is preserved when you relaunch.

## Customization

The ClickGUI supports extensive visual customization. From the HUD settings page, you can adjust:

- **Foreground color** - Change the text and icon colors.
- **Background color** - Set the background color of windows and panels.
- **Border color** - Customize the border color of UI elements.
- **Corner rounding** - Control how rounded the corners of windows appear.
- **Effects** - Toggle additional visual effects applied to the UI.

These options allow you to personalize the look and feel of Aoba to your preference.

## Navigation Pages

The ClickGUI is organized into several pages, each grouping different types of content.

### Modules

This page lists all available modules organized by category. Each module has a **gear icon** next to it. Clicking the gear opens a settings window 
for that specific module. Clicking the module name toggles it on or off.

Any installed Aoba addons will also have their modules displayed here, including any custom categories they define.

### Tools

A central location for GUI-based tools. Currently this includes the **AuthCracker** for offline servers, which allows you to test 
cracked accounts using a set of commonly used passwords.

### HUD

This page lets you manage on-screen HUD elements and GUI-specific settings.

- **HUD Elements** - A window lists all available HUD overlays. Click **(+)** or **(-)** to enable or disable them. Enabled HUD elements 
remain visible on screen even when the ClickGUI is closed. They can be moved and resized the same way as windows.
- **GUI Settings** - A separate window contains all settings related to the GUI appearance, including colors, effects, and the keybind for toggling the ClickGUI.
